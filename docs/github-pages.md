# GitHub Pages
GitHub Pages werden als Markdown geschrieben und mit Hilfe des static site generators jekyll gebaut.
* Editor: Visual Studio Editor
* Markdown Engine: Kramdown
* Theme: architect via jekyll-remote-theme (Plugin)
*
